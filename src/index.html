Lekki Store

This is an ecommerce site. Front end was made with React and styling was achieved with styled components. The backend was done with Node on the express framework, Postgres was the database of choice, ORM used is Sequelize. JWT was used for authentication, Bcrypt was used for password hashing. The user can create an account, login and place an order. The user can also view all previous orders.
